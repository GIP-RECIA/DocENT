Aide ENT > Changement de l'avatar de l'utilisateur

Chaque utilisateur de l'ENT peut se choisir un avatar qui sera affiché dans l'ENT en remplacement de l'avatar affiché par défaut. Le changement de l'avatar se fait dans le service "Mon compte ENT" de façon très simple.

Il convient tout d'abord d'accéder à "Mon compte ENT". Ceci se fait en cliquant sur l'avatar du bandeau ENT puis sur l'avatar qui s'affiche dans la petite fenêtre modale :

Cela se fait aussi depuis le menu  menu, en cliquant sur l'avatar affiché :

"Mon Compte ENT" affiche à l'utilisateur l'ensemble des informations qui le concernent et lui permet de changer certaines données, dont son avatar. Pour accéder à la modification de l'avatar, il suffit alors de cliquer sur l'avatar courant :

S'ouvre alors une nouvelle fenêtre (popup) qui permet à l'utilisateur de choisir le fichier image qui lui servira d'avatar.

Attention : il se peut que le navigateur empêche l'ouverture de la fenêtre popup et il faut alors autoriser l'ouverture de popups pour le site de l'ENT.

Une fois le fichier sélectionné, il conviendra de dimensionner l'avatar correctement pour qu'il puisse être utilisé dans l'ENT :

Si le fichier choisi ne convient pas, il est possible d'en choisir un autre en cliquant sur "Parcourir".
L'image du bas montre comment l'avatar sera vu dans l'ENT.
Une fois que tout est correct, il ne reste plus qu'à cliquer sur "Valider".

L'avatar est alors visible dans "Mon compte ENT" :

mais aussi dans le bandeau de l'ENT :

et dans le menu de l'ENT, accessible depuis menu :
