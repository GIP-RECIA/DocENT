Administration déléguée dans le Moodle de l'ENT

Les administrateurs délégués Moodle pour un établissement sont les membres du groupe ENT "Admin:Moodle:Administration locale de Moodle:<Établissement>".
Les administrateurs délégués peuvent, dans la catégorie correspondant à leur établissement :

  • gérer les sous-catégories de leur établissement
  • ranger les cours dans ces sous-catégories
  • supprimer les cours de leur établissement (par exemple quand un collègue est parti depuis longtemps et que le cours n'a plu lieu d'être).
