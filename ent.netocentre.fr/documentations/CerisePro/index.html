Mise à disposition de l'application

Le raccordement de CPro STI/Cerise Pro consiste en la mise en place d’un connecteur permettant d’établir une correspondance entre un utilisateur de l’ENT et un utilisateur de l’application. Cette correspondance va se faire en stockant dans la base de données de l’application l'identifiant ENT de l'utilisateur.
Conformément au SDET, l’ENT ne fournira aucune information personnelle dans le ticket CAS, ce qui implique que pour effectuer le rapprochement de tous les comptes (ENT ↔ Application), il est nécessaire de passer par l’importation d’un fichier CSV en plus des traditionnels fichiers XML issus de STS_EMP et de SIECLE.
Ce fichier CSV est mis à disposition des établissements par l’ENT et sera importé via la même interface d’import que les fichiers XML (les fichiers XML, puis les fichiers CSV).

Schéma de principe du rapprochement des comptes ENT dans Cerise Pro

Étape 1 : une personne autorisée à réaliser des exports dans la Gestion des Comptes ENT extrait un fichier formaté pour CPro.
Étape 2 : l'outil de Gestion des Comptes ENT constitue la liste des utilisateurs demandés.
Étape 3 : la personne qui a demandé la liste reçoit le fichier au format CSV.
Étape 4 : l'administrateur CPro importe dans CPro le fichier CSV pour réaliser le rapprochement des comptes.

Cette suite d'opérations est à réaliser pour les enseignants et pour les élèves à tour de rôle, le format de fichier étant différent.

Rapprochement des comptes dans Cerise Pro ou Cerise Pro STI (CPRO)

La correspondance entre un compte Cpro et un comptes ENT ne peut se faire que si celle-ci est unique. En cas de doublon, il sera nécessaire d'intervenir manuellement.
Pour les établissements qui souhaitent mettre en place le connecteur, voici les actions à réaliser :

  1. L’administrateur de CPro dans l’établissement et/ou l’administrateur ENT envoie un courriel à l’assistance (assistance-cpro@reseau-canope.fr) pour demander la mise en place du connecteur en précisant son RNE et la date à laquelle il souhaite la mise en œuvre ;
  2. L’assistance active les fonctions d’import des fichiers CSV issus de l’ENT et répond à l’établissement ;
  3. L’administrateur de l’application importe les fichiers CSV issus de l’ENT dans le module ""Importer les fichiers" ;
  4. À la date convenue :
    • a. l’administrateur de l’ENT crée la ressource permettant aux utilisateurs d’accéder à l’application Cerise pro/Cpro STI
    • b. l’assistance active le connecteur.
    • c. un échange téléphonique entre l’assistance et l’administrateur de l’application est souhaitable pour tester un compte enseignant ou élève en direct ;
  5. Dès que le connecteur est mis en place, la page de l’application n’est plus accessible, une redirection vers l’ENT est mise en place. En revanche, l’administrateur de l’application doit s’authentifier à partir de l’URL suivante en utilisant ses identifiants habituels :
    • https://www.cerise-pro.fr/admincerisepro pour l’application Cerise pro.
    • https://www.cpro-sti.fr/admincpro pour l’application Cpro STI.
  6. L’administrateur de l’application doit continuer à importer les fichiers SIECLE/STS_EMP dans le module Importer les fichiers pour mettre à jour les bases utilisateurs de l’application. Pour que les nouveaux utilisateurs puissent accéder à l’application, il est nécessaire que l’administrateur de l’ENT exporte des fichiers CSV de l’ENT pour les importer dans le module Importer les fichiers de l’application afin que celle-ci et l’ENT disposent d’informations communes pour le connecteur.
Contact assistance : assistance-cpro@reseau-canope.fr

Mise à disposition de l'application aux publics concernés dans l'ENT

Une fois le connecteur mis en place, l'administrateur ENT peuple le groupe Cerise Pro (Applications:Cerise Pro:Établissement) et/ou le groupe Cerise Pro STI (Applications:Cerise Pro STI:Établissement) avec les populations concernées.
